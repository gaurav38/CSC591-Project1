To compile
----------
Make sure the directories include and src are in the same directory as makefile
$ make

To compile for debug
--------------------
$make debug

To execute
----------
$./proj1 amazon.graph 100 amazon.output 0.1
This will create an output file amazon.output after running the algorithm for 100 iterations on the graph amazon.graph and apply threshold of 0.1


All coding and testing done on VCL. Centos 5.9 x64 and 6.4 x64 images
